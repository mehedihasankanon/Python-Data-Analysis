## Requirements 

- use command prompt terminal
- create a virtual environment first, running this in your terminal  
    ```python 
    python -m venv venv 
    ``` 
- activate the virtual environment by running in terminal: 
    ```python 
    venv\Scripts\activate
    ```
- if `(venv)` appears in your terminal, then it's activated successfully. 
- now, install the python libraries using this command in your terminal: 
    ```python
    pip install numpy, pandas, matplotlib
    ```
